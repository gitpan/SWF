NAME

SWFMovie_setNetworkAccess - sets SWF network permission

SYNOPSIS

#include <ming.h>

void SWFMovie_setNetworkAccess
(
	SWFMovie movie,
	int flag
);

PARAMETERS

SWFMovie movie
Not Documented.
int flag
Not Documented.

DESCRIPTION

If flag is set to 0, a localy loaded movie will be able to access the network For SWF >= 8: default is 0.
